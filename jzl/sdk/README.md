# 1 微网格开放接口

通过简单的get请求，从高速缓存中获取后评价数据


## 1.1 快速开始

可以在任何网络环境下，点击下面链接访问北京2017年4月数据：

http://api.imaoda.com/jzl/post_evaluate?month=201704&city=Beijing

同理，访问上海4月的数据：

http://api.imaoda.com/jzl/post_evaluate?month=201704&city=Shanghai

不过我们更推荐用城市code来访问，如访问北京：

http://api.imaoda.com/jzl/post_evaluate?month=201704&city=10100

> Beijing的城市code：10100

## 1.2 API接入规范

只需修改城市和月份，即可快速获取想要的数据：

![](http://www.imaoda.com/s/img/jzl/illu.png)

> 从2017年4月份开始有数据

**请求方式**

支持所有主流的方式访问，包括

- 浏览器直接访问
- 后台 get 请求
- 后台 post 请求
- 前端 异步 请求
- 前端 跨域 请求
- 支持 https 安全ssl请求
- wget 下载
- curl 下载

> https 安全请求主要应用于 ios系统、微信小程序，时延较普通http请求多20毫秒

# 2 其他说明 

**更新周期**

API按月份更新，每月1日可更新上月数据

> 由于不可抗因素，例如国庆下电，更新时间会顺延

**https请求**

目前https为试用，试用期截止2017年7月30日，后续如有需求，可考虑购买证书持久化

**接口调用频次**

目前接口调用未做限制，仅供项目组内部试用

**问题反馈**

使用过程中如有问题，欢迎随时反馈
